= ERDE

Lightweight ER-Diagram editor for Eclipse.


NOTE: Version 1.0.x is under development.


== Installation

eclipse 2020-12(java11)


[source]
----
dropins
 |-erde
    |-eclipse
       |-plugins
          |-erde-x.x.x.jar
----
